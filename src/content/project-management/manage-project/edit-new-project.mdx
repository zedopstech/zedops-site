## **How to Edit an Existing Project**

Update project details like name, status, client, or site locations directly from the **Projects Dashboard**.

---

### 🔹 **Access the Project Management Section**

1. From the **Home Page**, go to the **Top Navigation Bar**.
2. Hover over **Project Management** and click on **Manage Projects**.
3. The **Projects Dashboard** will display a list of all projects with key information and actions.

---

### ✏️ **Edit a Project**

1. Locate the project you want to update.
2. Click on either the **Project Name** or the **Edit (✏️) icon** (if available).
3. You’ll be redirected to the **Edit Project** page where all project details are pre-filled.

---

### 📝 **Update Project Details**

You can update any of the following fields:

- **Project ID** – Update if needed (optional, if editable).
- **Project Name** – Modify the project title.
- **Project Status** – Change the project’s current state.
- **Project Value** – Adjust the project’s financial value.
- **Currency** – Update the currency if required.
- **Start Date / End Date** – Modify the timeline using the date pickers.
- **Project Type** – Change the project type category.

---

### 👤 **Update Client**

- **Client Dropdown** – You can change the client assigned to this project.
- If the required client is missing:
    - Click **"Add New Client"** to go to the **Clients Directory** and create one.
    - Return to the project form and click **"Refresh"** to reload the client list.

---

### 📍 **Update Site Location(s)**

- Add, update, or remove **Site Locations** as needed.
- You can also enter new locations or modify existing ones.

---

### ✅ **Save Changes**

- After making all necessary updates, click the **"Save"** or **"Update Project"** button.

Your project details will now be updated and reflected in the **Projects Dashboard**.
