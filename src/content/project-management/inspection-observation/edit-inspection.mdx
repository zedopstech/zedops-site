## **How to Edit an Inspection at the Project Level**

Once an inspection is created, you can revisit it to make changes, add missing details, or correct entries before final submission or report generation.

1. From the **Home Page**, go to the **Top Navigation Bar**.

2. Select the project for which the inspection was created.
3. The list of inspections for that project will be displayed in the table view.
